Welcome to the GitHub site for "TEI and Digital Edition Production with LEAF Commons" at DH@Guelph 2026.

In addition to GitHub and GitHub Desktop (which we will use for sharing and publishing files), we will use the following software programs and platforms:

* Oxygen XML Editor [Download here](https://www.oxygenxml.com/xml_editor.html)
* [LEAF-Writer Commons](https://leaf-writer.leaf-vre.org/)
* [Dynamic Table of Contexts](https://dtoc.leaf-vre.org/)

_Do not_ pay for a Oxygen license -- you will only need the 30-day free trial license.
_Do not_ pay for a GitHug subscription. Use the free version.

You will need your GitHub credentials to log into LEAF-Writer and DToC. 

When you create your TEI files in Oxygen you will add the following CSS file link to your header.
